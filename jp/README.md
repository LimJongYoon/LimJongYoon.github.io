        <span class="portfolio-title">HapTug: Force-Feedback Haptic Device for Representation of Tugs in Virtual Reality</span>
        <span class="portfolio-middle"><strong>Jongyoon Lim</strong>, Yongsoon Choi<br>
        Electronics 11 (11), 1730</span><br>
        <span class="summary">この研究では、バーチャルリアリティ（VR）のための触覚デバイスが提案されています。提案された触覚デバイスは、VRデバイスのバンドルコントローラーにリンクされています。本研究で提案されたデバイスは、物体との衝突感ではなく、手で感じる触覚に焦点を当てた力フィードバックデバイスです。このデバイスは、ユーザーがツールを振るときに、その重さによる遠心力または慣性力を感じさせ、ユーザーがツールで物体を刺したり突いたりするときに反応力を感じさせます。リンク型の触覚デバイスによって生成される力は、仮想環境でツールを実際に使用する感覚が提供されるかどうかを判断するために測定され、検討されました。また、2種類の実験を実施してこの触覚を評価しました。最初に、ユーザー体験のためにVRコンテンツが作成され、30名の実験者を対象に、バンドルコントローラーとHapTugと呼ばれる触覚デバイスを現実感、没入感、楽しさの観点から比較する調査が行われました。調査結果は、バンドルコントローラーを単独で使用した場合と比較して、現実感、没入感、楽しさが向上したことを示しました。次に、41名の実験者が6回、各30秒間、仮想環境で自由にプレイしました。彼らは30秒間制御するためにボタンを押す必要があり、装着したデバイスは毎回無作為にオンとオフを繰り返しました。体験中に、再生時間、相互作用の数、および範囲が測定されました。その結果、デバイスが起動したとき、実験者は時間が短いと感じ、広範な動きを示しました。このように、提案されたリンク型触覚デバイスは、ユーザーが仮想オブジェクトと相互作用するときに手に出入りする感覚を提供し、ツールを使っている感覚を効果的に伝えることが証明されました。さらに、実験で3つの可能性が見つかりました。以前のVR体験時間は触覚使用の満足度とはほとんど関係がなく、ユーザーは触覚の現実感に対する期待が低く、ユーザーは触覚デバイスとより積極的に相互作用することが分かりました。</span> <a href="javascript:void(0);" onclick="toggleText(this)">View Details</a>
